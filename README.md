# Ubuntu Ceph demo

## About

An unattended Ceph installation demo on four virtualized Ubuntu nodes. It took 16 minutes if the Ubuntu ISO file was downloaded.

## Run
```
make -j
```

## Test
Tested on Arch Linux on 2023-04-08.
